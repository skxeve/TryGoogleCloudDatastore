# Datastore via PHP

```
$ dev_appserver.py app.yaml
```

## var.yaml

```
$ cat var.yaml

    APP_DATASTORE_PROJECT_ID: "project_id_hoge_fuga"
    APP_DATASTORE_NAMESPACE_ID: "try-gae"
```
